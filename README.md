# B2B Dealer Automation Panel (Simulation)

This is a simulation of a B2B dealer panel designed to streamline:
- Dealer onboarding
- SAP creation
- UCM counter mapping
- Dealer payouts

## Purpose
This project was created to:
- Visualize system workflows
- Align business, product, and engineering teams
- Demonstrate process automation possibilities

## Key Features
- Structured dealer onboarding flow
- Workflow-based UI simulation
- Simplified operational process mapping

## Tech Used
- HTML, CSS (basic)
- GitHub Pages (for hosting)

## Note
This is a simulation/prototype for workflow demonstration purposes only and does not represent any production system.
